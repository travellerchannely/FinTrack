# FinTrack — Фінансовий облік

Веб-сервіс для фінансового обліку: Node.js + Express бекенд, JSON-файлова БД.

## Запуск локально
```bash
npm install
npm start
# Відкрити http://localhost:3000
```

## Деплой на Render.com (безкоштовно)
1. New → Web Service → підключіть цей репозиторій
2. Build Command: `npm install`
3. Start Command: `node server.js`
4. Plan: Free
5. Додайте Disk: Mount Path `/opt/render/project/src/db`, Size 1 GB
